lsmidi
======

Simple command to list midi devices available

Usage
-----

```
$ npm install -g lsmidi
$ lsmidi

MIDI devices

input
  [0] USB Axiom 49 Port 1
  [1] USB Axiom 49 Port 2
  [2] E-MU XMidi1X1 Tab In

output
  [0] USB Axiom 49 Port 1
  [1] E-MU XMidi1X1 Tab Out
```
